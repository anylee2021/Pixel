# Build and Run
`git clone https://github.com/pixel-Teee/Pixel.git`

`cd Pixel`

`git submodule update --init --recursive`

`mkdir build`

`cd build`

`cmake .. -DCMAKE_GENERATOR_PLATFORM=x64 -DCMAKE_BUILD_TYPE=Debug`
